How to reset mysal password

Step 1 stop the mysql server if its running

/etc/init.d/mysqld stop

Step 2 Run the mysql in safe mode using skip grant table option.

mysqld_safe --skip-grant-tables &

step 3 login into mysql using without password

mysql –u root  press enter

step 4 change the password for root user.

> use mysql ;
> update user set password=PASSWORD("NEW-ROOT-PASSWORD") where User='root';
>flush privileges;
>quit;

Step 5 stop the mysql.

/etc/init.d/mysqld stop

Step 6 start the mysql in normal mod.

/etc/init.d/mysqld start

Step 7 now login in mysql with user root and password what u reset.

#Mysql –u root –p
# enter password

Sendmail, Squirrelmail and Dovecot

Package needed

sendmail-cf-8.13.8-2.el5
sendmail-8.13.8-2.el5
squirrelmail-1.4.8-4.el5
dovecot-1.0-1.2.rc15.el5

configuration files

For sendmail

all files present in /etc/mail/ directory

/etc/mail/sendmail.mc
/etc/mail/sendmail.cf 
/etc/mail/access
/etc/aliases

For Squirrelmail

Go to directory

/usr/share/squirrelmail/config/

And run script conf.pl

For dovecot

/etc/dovecot.conf

Sendmail configuration

Edit the file sendmail.mc and comment the below line using dnl parameter as shown below.

dnl DAEMON_OPTIONS(`Port=smtp,Addr=127.0.0.1, Name=MTA')
dnl # DAEMON_OPTIONS(`Port=smtp,Addr=127.0.0.1, Name=MTA')

after that run the below command to right the changes in sendmail.cf file.

m4 /etc/mail/sendmail.mc > /etc/mail/sendmail.cf
make

then edit the file /etc/aliases

and make the entry at the bottom of the file as given below.

vi /etc/aliases 
amar1:          amar1@amar.com
amar:           amar@amar.com

here amar and amar1 are the user having mail address amar@amar.com and amar1@amar.com respectively,

After that go to directory /etc/ using cd /etc/
And run the command newaliases which crate the aliases db file we have to run this every time we edit the /etc/aliases file

Now we have to configure relay for this mail server

Edit the file /etc/mail/access and place the below line.

vi /etc/mail/access
172.16.34.0 RELAY

le="text-align: left;">It will relay all the mail coming from 172.16.34.0 network.

After that go to /etc/mail directory and run the below command for creating the access.db file.

makemap hash access.db < access

Or

makemap hash /etc/mail/access.db < /etc/mail/access

we have to run this command to update the access.db command every time we place the relay entry in access file.

Squirrel mail configuration

Just go to /usr/share/squirrelmail/config/ directory
And run the config.pl script and configure the squirrel mail as per your setup.

Dovecot configuration

Edit the file

vi /etc/dovecot.conf

And uncomment the below line

protocols = imap imaps pop3 pop3s

basic configuration part is completed

Now restart the services as below

/etc/init.d/sendmail restart
/etc/init.d/dovecot restart
/etc/init.d/httpd restart

Steps :-

1> Create key file.

openssl rand -base64 753 > repkey.txt

2> Add entries in /etc/mongod.conf on Master Server.

logpath=/opt/mongo/current/logs/mongod.log
logappend=true
fork = true
port = 27017
dbpath=/opt/mongo/current/db
pidfilepath = /opt/mongo/current/mongod.pid
keyFile = /opt/mongo/repkey.txt
master = true
source = mongodb1

3> Add entries in /etc/mongod.conf on Slave Server.

logpath=/opt/mongo/current/logs/mongod.log
logappend=true
fork = true
port = 27017
dbpath=/opt/mongo/current/db
pidfilepath = /opt/mongo/current/mongod.pid
keyFile = /opt/mongo/repkey.txt
slave = true
source = mongodb1

Now restart Mongodb

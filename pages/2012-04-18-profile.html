Cisco Pix Implementation
The project is to deploy the security solution for protecting the network from outside threat. For this 2 PIX 515e series, firewall devices in failover configuration mode are being used. One PIX firewall devices is used as primary and other as standby for the redundancy. Three interfaces for both the PIX will be used for outside, inside and stateful failover.
We design our Pix in such a way to understand our network (LAN/WAN) requirement and act accordingly to proposed design. i.e.

  • Only HTTP/HTTPS/POP3 access from outside to inside for Internet users. All Services restricted from outside to inside.
  • Only Mail Server access from outside to inside through spectra net for mobile Users.
  • All  Services allowed from inside to outside.
  • Back-up internet connectivity through ISDN from VSNL for inside users.

For Remote Locations: 
For emote locations, we installed Cisco Pix 515e firewall without failover configuration.

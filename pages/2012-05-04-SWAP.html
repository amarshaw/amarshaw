Increase the size of Swap Memory

Login to system with root user
Check in your "/" filesystem that you have sufficient space. If not then extend "/" file system

#lvextend -L+2G /dev/mapper/root-lv
#xfs_growfs /

Now Adding 2 GD more swap memory of system.
2 GB = 1024*2048MB = 2097152 KB
Use dd command to create swapfile

dd if=/dev/zero of=/swapfile1 bs=1024 count=2097152

Where,
if=/dev/zero : Read from /dev/zero file. /dev/zero is a special file in that provides as many null characters to build storage file called /swapfile1.
of=/swapfile1 : Read from /dev/zero write stoage file to /swapfile1.
bs=1024 : Read and write 1024 BYTES bytes at a time.
count=2097152 : Copy only 2097152 (2 GB) BLOCKS input blocks.

Set Up a Linux Swap Area

mkswap /swapfile1

Set the ownership of /swapfile1 to root:root and permission 0600

# chown root:root /swapfile1
# chmod 0600 /swapfile1

Now activate this Swap partition with swapon command

# swapon /swapfile1

Now add entry for new swap area in /etc/fstab file, so that its autometically activated during system reboot.

# vi /etc/fstab
/swapfile1 swap swap defaults 0 0
#mount -a

Now check the swap memory status

#free -m

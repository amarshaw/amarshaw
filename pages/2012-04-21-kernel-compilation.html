Kernel Compilation

 Download the latest kernel from http://www.kernel.org/

  1. Copy it on /tmp or any where as per space availability.
  2. extract it on /usr/src/kernel/ folder using command tar –xjvf kernel file.
  3. take the backup for /boot files
  4. check the directory permission on that new extracted directory it should be like that  
    “drwxr-xr-x   19 root     root         4096 Apr 06 08:29 directory name”
  5. If it’s not like that then change it using chmod command.
  6. Then check whether gcc compiler is installing or not if not installed it using yum if yum is not available install it using rpm.
  7. After installing the gcc go to the extracted directory of kernel which will be at /usr/src/kernel/directory_name.
  8. run the command “make menuconfig” for configuring the kernel change the setting if you are expert in it other wise select the default option and save it and make exit from it, when exit from that configuration window it will create a hidden file name .config at the same directory.
  9. After then run the command “make dep” This command builds the tree of interdependencies in the kernel sources. These dependencies may have been affected by the options you have chosen in the configure step.    
  10. Then run the command “make zImage” it will create the image file for kernel.
  11. Then run the command “make modules” it will build the module if it takes longer time depending on how much module do u have.
  12. In this step will install all modules using command “make modules install” .This will copy all the modules to a new directory, "/lib/modules/a.b.c" where a.b.c is the kernel version.
  13. after that we will install the kernel using command “make install” it will copy the below files to /boot location
vmlinuz-2.6.27.49
System.map-2.6.27.49
initrd-2.6.27.49.img

15 files must be present on /boot location are :-

vmlinuz-2.6.27.49
System.map-2.6.27.49
initrd-2.6.27.49.img
config-2.6.27.49
Config-2.6.27.49

For this file go to location /usr/src/kernel/extracted_kernel_dir/ and copy the file .config what we have created in step 9 using command

cp .config /boot/config-2.6.27.49

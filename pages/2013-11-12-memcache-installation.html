Memcache

Prerequisite : libevent on shell

libevent on shell Installation

wget https://github.com/downloads/libevent/libevent/libevent-2.0.21-stable.tar.gz
tar xfz libevent-1.4.8-stable.tar.gz
cd libevent-1.4.8-stable
./configure
make
sudo make install

Now we need to create a symlink for libevent

ln -s /usr/local/lib/libevent-1.4.so.2 /usr/lib

Memcached Installation :- 

wget http://danga.com/memcached/dist/memcached-1.4.15.tar.gz
tar xfz memcached-1.2.6.tar.gz
cd memcached-1.2.6
./configure
make
make install

Now Run Memcached as a daemon.

memcached -d -m 1024 -u root -l 127.0.0.1 -p 11211 on shell

d = daemon
m = memory
u = user
l = IP to listen to
p = port

SUDO

Install sudo rpm
Type visudo to edit /etc/sudoers file

vi /etc/sudoers
## Allow root to run any commands anywhere
## Allow root to run any commands anywhere
root    ALL=(ALL)       ALL
shaw    ALL=(ALL)       ALL
amar    ALL=/etc/aaa,/bin/vi,/bin/rm

Add user permission like above example.
Test it by login through amar / shaw user.

sudo vi /etc/aaa
sudo rm /etc/aaa
sudo /sbin/reboot

Shaw user will do the reboot as only shaw user have the full permission in this scenario.

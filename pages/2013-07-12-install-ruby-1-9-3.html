Install Ruby 1.9.3 from RVM.

See requirements for RVM

rvm requirements

Install RVM

\curl -L https://get.rvm.io | bash
rvm install 1.9.3

Install Ruby 1.8.7 from RVM

rvm install 1.8.7

To see the installed ruby version.

rvm list

To see the ruby version 

ruby -v

Set Default Ruby version

rvm use 1.9.2
Or
rvm use 1.9.3 --default

Install Ruby 1.9.3 from tar file

wget ftp://ftp.ruby-lang.org/pub/ruby/1.9/ruby-1.9.3-p448.tar.gz
tar -xvf ruby-1.9.3-p448.tar.gz
cd ruby-1.9.3-p448/
./configure
make
make install

Install libyaml from tar file

tar zxf yaml-0.1.4.tar.gz
cd yaml-0.1.4
./configure
make
make install

Install libyaml from rvm

rvm pkg install libyaml
